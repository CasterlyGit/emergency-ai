// Latency

<800 ms to first visible action via streaming. Prompt-cached system prompt cuts TTFT from ~600ms to ~120ms after the first hit in a 5-min TTL window.

// Jurisdiction

6 cities seeded. Each has a multi-KB markdown brief on laws (Good Samaritan, drug amnesty), emergency numbers, and cultural context. The model is grounded — not vibes.

// Discipline

Strict pydantic schema. No wall of text. Urgency → ordered actions → who to call → what to avoid. The schema is both UI contract and forcing function for the model.
